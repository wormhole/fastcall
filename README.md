## 一、简介
`fastcall`是一款`java`开发的低配置、无侵入的`RPC`框架，采用了类似`dubbo`的注解风格，支持`ZooKeeper`服务注册中心，对于`Spring`和`非Spring`应用都完美支持，并提供`fastcall-spring-boot-starter`，支持完全自动配置。

## 二、支持情况
|序列化方式|注册中心|协议|
| ---- | ---- | ---- |
| - [x] json    | - [x] zookeeper | - [x] fastcall |
| - [ ] protobuf| - [ ] redis     | - [ ] http     |
| - [ ] msgpack | - [ ] Multicast | - [ ] RMI      |
